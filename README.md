# homebrew-tap

Homebrew tap for [rdc](https://github.com/mrtnzlml/rossum-deployment-manager-experiment) and related tools.

## Install

```sh
brew install mrtnzlml/tap/rdc
```

## Tools

| Formula | Description | Source |
|---|---|---|
| `rdc` | Rossum Deployment as Code -- CLI for snapshotting and deploying Rossum.ai configurations | [mrtnzlml/rossum-deployment-manager-experiment](https://github.com/mrtnzlml/rossum-deployment-manager-experiment) |

## Updates

The `rdc` formula is regenerated automatically on each `v*` tag of the source repo by the release workflow there. Manual edits to `Formula/rdc.rb` will be overwritten by the next release.
